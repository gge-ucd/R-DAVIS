Resources

There are many resources online and in print. This is by no means a comprehensive list, but hopefully a launching pad for some very useful website and books. All of these examples provide code in same shape or form.

Datasets

This is a list of most of the data sources used in this course.

  • From The Portal Project Teaching Database
    • Portal_rodents_19772002_scinameUUIDs.csv [raw trapping data]
    • species.csv [species id metadata]
    • surveys.csv [trapping data by record id]
    • surveys_complete.csv [cleaned trapping data by record id]
    • combined.csv [trapping data merged with record id metadata]
    • plots.csv [plot type metadata]
  • From Data Carpentry: Genomics Lesson
    • sequencing_results_metadata.txt [metadata from DNA sequencing]
  • From Data Carpentry: R for Data Analysis and Visualization of Ecological Data
    • survey_data_spreadsheet_messy.xls [example of untidy data]
  • From Software Carpentry: R for Reproducible Scientific Analysis
    • continents.RDA [gapminder continent-level data]
    • wide_eg.csv [blood albumin levels]
    • stocks.tsv [restaurant stock data]
    • gapminder.csv [country characteristics by year]
  • From ANGUS: Analyzing High Throughput Sequencing Data
    • rnaseq.zip [zipped folder of RNA sequence data]
  • From Data wRangling for Ecologists
    • 2001_mauna_loa_met_data.zip [raw meteorological station data from Mauna Loa]
    • 2001 Mauna Loa meteorological station data [tidy format of minute-scale meteorological data from Mauna Loa Observatory]
    • solinst temp/stage logger data (full) [pressure transducer data logger data from NF American River]
    • solinst A [partial pressure transducer data logger data from NF American River]
    • solinst B [partial pressure transducer data logger data from NF American River]
    • solinst 2015
